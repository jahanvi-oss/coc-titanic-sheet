# coc-titanic-sheet
Titanic Sheet
Overview

Titanicsheet is a Google Sheet project containing a dataset of historical passenger records, modeled after the famous Titanic dataset. It includes information such as passenger class, age, fare, family relations, and survival status — useful for data exploration, visualization, and machine-learning exercises.

Files

titanicsheet — the main Google Sheet containing multiple tabs:

Raw Data — original passenger dataset.

Analysis — summary statistics and exploration.

Pivot Table 1 / 2 / 3 — pivot-table summaries for quick insights.

Dashboard — aggregated charts and metrics.


Dataset Structure

Each row represents a single passenger.
Key columns include:

PassengerId — unique identifier

Survived — survival flag (0 = No, 1 = Yes)

Pclass — passenger class (1 = 1st, 2 = 2nd, 3 = 3rd)

Name, Sex, Age

SibSp, Parch, Ticket, Fare, Cabin, Embarked

Usage

Open the titanicsheet Google Sheet

Make a copy to your Google Drive if you plan to edit.


Use it for:

Exploratory Data Analysis (EDA)

Data cleaning and preprocessing

Machine-learning model training

Visualization dashboards


Notes

Some columns (e.g., Age, Cabin, Embarked) may contain missing values.

The dataset is ideal for educational and analytical use.

Attribution is appreciated if you share results based on this sheet.
